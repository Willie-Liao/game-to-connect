# game-to-connect

Quick mini-games with minimal rules and a shared win counter—no quizzes, no heavy scoring. The winner draws a safe question from the deck; the other answers aloud. Premium, intimate, and never a test.

## Run locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env.local` and set `GEMINI_API_KEY` if required
3. Run the app: `npm run dev`
